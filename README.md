# Tada! Frontend

A beautiful todo list application built with React and Vite.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Animated purple-to-blue gradient background
- RESTful API integration

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## API Configuration

Make sure your backend API is running on `http://localhost:8080/api/todos` or update the API URL in `src/todoApi.js`.
